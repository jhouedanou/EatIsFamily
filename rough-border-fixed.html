Cette card démontre que tout fonctionne avec un contenu variable. La hauteur s'adapte, les bords restent rugueux, et surtout le texte est parfaitement lisible sans aucune déformation. C'est la magie du ::before / ::after avec z-index.

📋 Le pattern à retenir

/* La card = juste un conteneur positionné */
.card {
  position: relative;
  padding: 32px;
  width: fit-content;
}

/* Le fond déformé = pseudo-element */
.card::before {
  content: '';
  position: absolute;
  inset: 0;
  background: #fff;
  border-radius: 16px;
  filter: url(#rough-border);
  z-index: 0;  /* derrière */
}

/* Le contenu = au-dessus, PAS de filtre */
.card > * {
  position: relative;
  z-index: 1;  /* devant */
}
